# JavaScript AMA – Questions and Answers

1. **What is `setTimeout` in JavaScript?**  
A function that executes code after a specified delay (in milliseconds) without blocking the program.

2. **What is an object in JavaScript?**  
A collection of key–value pairs that can store data and functions.

3. **Explain scope in JavaScript.**  
Scope determines where variables are accessible:  
- Global → anywhere  
- Function → inside the function  
- Block (`let`, `const`) → inside `{ }` only

4. **What are the different types of datatypes in JavaScript?**  
- Primitives: `Number`, `String`, `Boolean`, `Null`, `Undefined`, `Symbol`, `BigInt`  
- Non-primitives: `Object` (arrays, functions, dates, etc.)

5. **What is JavaScript used for?**  
To make websites interactive and dynamic, both in browsers and servers (Node.js).

6. **What is the difference between `let`, `var`, and `const`?**  
- `var` → function-scoped, can be redeclared  
- `let` → block-scoped, can be reassigned  
- `const` → block-scoped, cannot be reassigned

7. **What is a function in JavaScript?**  
A reusable block of code that performs a specific task.

8. **What is an arrow function?**  
A shorter syntax for functions introduced in ES6.

9. **What is a callback function?**  
A function passed as an argument to another function and executed later.

10. **What are array methods?**  
Built-in functions to manipulate or process arrays, e.g., `.map()`, `.filter()`, `.reduce()`.

11. **What is the purpose of `.map()`?**  
To create a new array by transforming each element of an existing array.

12. **What is the purpose of `.filter()`?**  
To create a new array containing only elements that meet a condition.
