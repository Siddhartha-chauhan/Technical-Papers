# AMA Q&A 

1. **Explain ACID properties in SQL.**  
   ACID stands for **Atomicity, Consistency, Isolation, Durability**.  
   - **Atomicity:** A transaction is all or nothing.  
   - **Consistency:** Ensures data remains valid after a transaction.  
   - **Isolation:** Each transaction runs as if it’s the only one.  
   - **Durability:** Once committed, data is permanently saved.

2. **What are array methods in JS?**  
   Common JavaScript array methods include:  
   - `push()` – add to end  
   - `pop()` – remove from end  
   - `shift()` – remove from start  
   - `unshift()` – add to start  
   - `map()` – transform array items  
   - `filter()` – filter elements  
   - `reduce()` – accumulate values  
   - `find()` – locate a value  
   - `slice()` – copy part of array  
   - `splice()` – add/remove items

3. **What is the purpose of GROUP BY clause?**  
   GROUP BY groups rows based on a column's value and is used with aggregate functions like COUNT, SUM, AVG.

4. **What are different ways to add color in CSS?**  
   - Named colors (`red`, `blue`)  
   - Hex codes (`#ff0000`)  
   - RGB (`rgb(255,0,0)`)  
   - RGBA (`rgba(255,0,0,0.5)`)  
   - HSL (`hsl(0,100%,50%)`)  
   - HSLA (`hsla(0,100%,50%,0.5)`)

5. **What is the purpose of the `<meta>` tag?**  
   The `<meta>` tag provides metadata like character set, viewport settings, page description, keywords, and SEO information.

6. **Difference between `<script>` in `<head>` and `<body>`?**  
   - In **head**: Script loads before the page renders, may block loading.  
   - In **body**: Script loads after content, improves performance and speed.

7. **What is Flexbox used for?**  
   Flexbox is used for efficient layout, alignment, and spacing in a single axis (row or column). It helps create responsive layouts easily.

8. **What does `display: none;` do?**  
   It **hides** the element completely — it takes no space in the layout and is not visible.

9. **Difference between `==` and `===` in JS.**  
   - `==` compares values **after type conversion** (loose equality).  
   - `===` compares **value and type** (strict equality).

10. **What are media queries?**  
    CSS media queries allow you to apply styles conditionally based on screen size, device type, orientation, etc.  

11. **What is a dictionary in Python?**  
    A dictionary is an unordered, mutable collection of key-value pairs.  
    Example:  
    ```python
    user = {"name": "Sahil", "age": 22}
    ```
