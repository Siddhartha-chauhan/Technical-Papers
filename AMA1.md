
## 1. What is PIP in Python?

`pip` is the Package Installer for Python. It installs, updates, and
manages external Python libraries from PyPI.

Example:

``` bash
pip install numpy
pip list
```

## 2. Why do we use `<span>` tag in CSS?

`<span>` is an inline tag used to style or manipulate a small part of
text without breaking the line.

Example:

``` html
<span class="highlight">word</span>
```

## 3. How can you return a range of characters in a string?

Using **slicing** in Python:

``` python
text = "HelloWorld"
text[0:5]  # Hello
```

## 4. What are empty elements in HTML?

Tags that do not contain content and have no closing tag.

Examples: `<br>`, `<img>`, `<hr>`, `<input>`, `<meta>`

## 5. Difference between list and tuple in Python?

  List        Tuple
  ----------- -----------
  Mutable     Immutable
  Uses \[\]   Uses ()
  Slower      Faster
  

## 6. Purpose of indentation in Python?

Indentation defines code blocks and replaces `{}`. Incorrect indentation
causes errors.

## 7. What is the purpose of a primary key?

It uniquely identifies each row in a table. Cannot be NULL and must be
unique.

## 8. Difference between WHERE and HAVING?

  WHERE             HAVING
  ----------------- -------------------
  Filters rows      Filters groups
  Before GROUP BY   After GROUP BY
  No aggregates     Allows aggregates

## 9. Difference between `<div>` and `<span>` in HTML?

  `<div>`           `<span>`
  ----------------- ---------------------
  Block-level       Inline
  Used for layout   Used for small text
  Breaks line       Does not break line

## 10. Purpose of `<head>` in HTML?

Contains metadata, title, CSS links, JS files, and SEO information.

## 11. Difference between class and id in CSS?

  Class               ID
  ------------------- --------------------
  Reusable            Unique
  Uses `.`            Uses `#`
  Lower specificity   Higher specificity

## 12. What is the CSS Box Model?

The box model describes the structure of every HTML element:

-   Content\
-   Padding\
-   Border\
-   Margin

Example:

``` css
.box {
  margin: 10px;
  padding: 20px;
  border: 2px solid black;
}
```
